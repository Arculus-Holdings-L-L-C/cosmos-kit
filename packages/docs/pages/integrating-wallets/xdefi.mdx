# How to Add CTRL Wallet to CosmosKit

There are two packages for ctrl

- `@cosmos-kit/ctrl`
- `@cosmos-kit/ctrl-extension`

`@cosmos-kit/ctrl` export all available ctrl wallets (currently only extension available), while if you only want to add a particular one, choose `@cosmos-kit/ctrl-extension`

> Note: all these packages export `wallets` and it's an array of `MainWalletBase`

Take `@cosmos-kit/ctrl` for example

## add `@cosmos-kit/ctrl`

```
yarn add @cosmos-kit/ctrl
```

## import the wallets

```js
import { wallets as ctrl } from "@cosmos-kit/ctrl";
```

## add to your provider

```js
function MyCosmosApp({ Component, pageProps }: AppProps) {
  return (
      <ChainProvider
        chains={chains}
        assetLists={assets}
        wallets={[...ctrl]}
        walletConnectOptions={...} // required if `wallets` contains mobile wallets
      >
        <Component {...pageProps} />
      </ChainProvider>
  );
}

export default MyCosmosApp;
```
